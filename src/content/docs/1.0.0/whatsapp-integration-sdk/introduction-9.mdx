## Integration Disconnect

Customer can ask the following during disconnect,

1. Phone number disconnect from whatsApp
2. Release phone number from BSP

Steps:

1. Channel deletion - [link](/1.0.0/portal/zohocorp/manual/sdk-reference/article/activate-deactivate-and-delete-channel#_Toc1dg753g4qr5n)

2. This will disconnect the phone number from whatsApp.

3. Options are available to have the conversations or delete channel along with conversations.

4. WABA removal

5. This will stop the process of incoming messages.

6. 2FA disable

7. This is done only if customer wants to use another BSP

8. Customer can directly go delete phone number from the WABA in meta console

9. Or we can manually disable 2FA for the phone number in meta console.

## BSP Migration

WhatsApp phone number migration enables migrating a registered phone number from one WhatsApp Business Account  (WABA) to another. Using this, a business can easily switch their phone number from an active WABA created in other business-service-providers to a new or an existing one created in Zoho.

Reference [link](/1.0.0/portal/zohocorp/team/zohoim/manual/sdk-reference/article/whatsapp-bsp-migration-1)
