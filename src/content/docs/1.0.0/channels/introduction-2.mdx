A channel refers to the platform or medium through which a message is sent or received. In IM a channel is created using configuration parameters, refer to the settings that can be adjusted to customize the behaviour of the channel.Each Integration Service account requires an IM Channel to be created with the necessary configuration parameters.

OAuth Scopes: ZohoIM.channels.ALL
